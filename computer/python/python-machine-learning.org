Python在机器学习领域有好多库可以使用，所以在学习机器学习的过程中对python加以了解是必要的。本文是我在学习python和机器学习过程中的一些学习资源和经验的总结。

* 别人的方法
  [2017-03-05 Sun 21:48] 

  一般，进入一个新的知识领域之前，我都会百度和谷歌一些别人的经验，所谓“站在巨人的肩膀上”。

  [[http://python.jobbole.com/84326/][利用Python，四步掌握机器学习]] 给出了从Python到机器学习的一系列资源，包括Python入门，机器学习的相关库。[[http://python.jobbole.com/84108/][七步精通Python机器学习]] 是同一网站的另一篇博文。虽然这两篇有点标题党，但文中的链接还是值得参考的。

  

Github上的资源一定要用起来，搜搜 *Python* 看看有哪些宝贝。
  

* 学习Python
  [2017-03-07 Tue 22:37]

  因为我学习Python的时候，Python版本已经到了3.5，所以不用纠结用不用学习2.7，果断选择3.5。最初的Python启蒙是从 [[http://www.liaoxuefeng.com/wiki/0014316089557264a6b348958f449949df42a6d3a2e542c000/001431658427513eef3d9dd9f7c48599116735806328e81000][廖雪峰的博客]] 开始的。本来想从《Learning Python the hard way》开始，但是只有2.7版本的，果断pass。这里给处于纠结到底是2.7还是3.X的后来者提个建议：除非是工作需要，直接学习3.X。Python只是一个工具，掌握Python是学习机器学习理论的附属品，没必要对其刨根问底。

  学习Python，
  1. 首先从其官方文档入手，[[https://docs.python.org/3.5/tutorial/index.html][英文版地址]] [[http://docspy3zh.readthedocs.io/en/latest/tutorial/][中文版地址]] 
  2. 然后Python最佳实践指南， [[http://docs.python-guide.org/en/latest/?utm_campaign=Manong_Weekly_Issue_10&utm_medium=EDM&utm_source=Manong_Weekly][英文版地址]] [[http://pythonguidecn.readthedocs.io/zh/latest/][中文版地址]] 
  3. 《Learning Python》可以找到电子版 
  4. 《Python 核心编程》可以找到电子版
     
  学习以上资料，挑选最新版本，都支持Python3.X
