#+TITLE: 交换 ubuntu 的 ctrl 和 caps
#+AUTHOR:    emacsun
#+EMAIL:     emacsun@163.com
#+DATE:
#+DESCRIPTION:
#+KEYWORDS:
#+CATEGORIES:
#+SETUPFILE: ~/.emacs.d/org-templates/emacs-level2.org

#+begin_example
sudo vi /etc/default/keyboard
#+end_example


修改下面这一行：（改变 capslock 为 ctrl）

#+begin_example
XKBOPTIONS="ctrl:nocaps"
#+end_example

或者可以交换 capslock 和 ctrl

#+begin_example
XKBOPTIONS="ctrl:swapcaps"
#+end_example


最后：

#+begin_example
sudo dpkg-reconfigure keyboard-configuration
#+end_example
