* Emacs theme gallary
尽管年老，Emacs也没有停止追求美。众多的Emacs拥趸为Emacs打造不同风格的主题。在 [[https://emacsthemes.com/index/1.html][这里]] 和 [[https://pawelbx.github.io/emacs-theme-gallery/][这里]] 保存了一些非常漂亮的Emacs主题。另外， [[https://draculatheme.com/emacs/][Dracula]] 也是一款比较漂亮的主题。

* spacemacs切换主题

我比较喜欢Solarized主题。这个主题有两套一套淡色（solarized-light）的一套深色(solarized-dark)。另外一个比较令我喜欢的是monokai。在spacemacs里可以通过 =spc T s= 来切换主题。

如果遇到自己比较喜欢的主题可以在 =init.el= 的 =dotspacemacs-themes= 变量列表中保存。这样 =spc T s= 跳出的列表中就会有你喜欢的主题供选择。目前我的列表是：
#+begin_example
   dotspacemacs-themes '(
                         monokai
                         solarized-dark
                         monokai
                         spacemacs-dark
                         spacemacs-light
                         solarized-light
                         leuven
                         zenburn)
#+end_example

* 美化Emacs

[[http://www.modernemacs.com/][Modern Emacs]] 为Emacs提供了一些列比较系统的美化，包括各种数学符号的显示，Eshell的显示，Magit的显示。整个Modern Emacs看起来非常赏心悦目。（插播一下，这个博客的主题叫做 [[https://draculatheme.com/emacs/][hugo academic]] 是我比较喜欢的一款博客主题。）

我个人并不喜欢这些过度优化，我还是更钟情最简单直接的文本。萝卜青菜各有所爱，喜欢的朋友可以移步去看看。
