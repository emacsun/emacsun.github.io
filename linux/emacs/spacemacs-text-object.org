* 简介
spacmeacs支持的 *操作符+操作对象* 这样的快速操作，这种组合方式允许快速的执行编辑操作，而且双手可以长期停留在键盘 =homeline= 上。
  
* spacemacs中支持的操作符
  spacemacs中支持如下操作符
  #+CAPTION: Spacemacs中支持的操作符
  #+NAME: tab:20170226operators
  #+ATTR_HTML: :border 1 :rules all :frame border :align center
  #+ATTR_LATEX: :align center
| 操作符 | 含义       |
|--------+------------|
| d      | 删除       |
| c      | 修改       |
| v      | 选择       |
| y      | 复制       |
| g~     | 大小写翻转 |
| gu     | 转换为小写 |
| gU     | 转换为大写 |

* spacemacs中支持的操作对象
