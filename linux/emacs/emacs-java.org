操作系统 Ubuntu. Java 版本 Java 8. 使用 Emacs + JDEE.
* Java 版本
#+begin_example
java -version
#+end_example
输出
#+begin_example
java version "1.8.0_131"
Java(TM) SE Runtime Environment (build 1.8.0_131-b11)
Java HotSpot(TM) 64-Bit Server VM (build 25.131-b11, mixed mode)
#+end_example
具体如何在 Ubuntu 上安装 Java [[http://blog.csdn.net/sh21_/article/details/52751367][见这里]] 或者 Google。

安装好 Java，记得设置 =classpath=
#+begin_example
export CLASSPATH = .:$CLASSPATH:/home/cliyh/zorg/coding/TIJ/
#+end_example
需要注销才能生效。
* 使用 JDEE
** 安装 JDEE

   通过 melpa 安装

** 安装 jdee-server
#+begin_tikzinstance
git clone https://github.com/jdee-emacs/jdee-server.git
cd jdee-server
mvn -Dmaven.test.skip=true assembly:assembly
#+end_tikzinstance
没有 =mvn= 命令，安装 =maven= .

#+begin_example
sudo apt install maven
#+end_example
继续执行：
#+begin_example
mvn -Dmaven.test.skip=true assembly:assembly
#+end_example
会从 maven 的官网下载很多东西，要等一会，漱口刷牙之后还没有下载完成，看情况是有很多小文件。结束之后 =jdee-server= 文件夹下有一个子文件夹名字是 =target= 。 在 =init.el= 中，把 =jdee-server-dir= 设置成 =target= 所在位置。
